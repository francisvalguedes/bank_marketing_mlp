Bank Marketing - A Multilayer Perceptron (MLP) Approach

By: Francisval Guedes , Hareton Gomes, Under the guidance: Prof. Ivanovitch

The data is related with direct marketing campaigns of a Portuguese banking institution. The marketing campaigns were based on phone calls. Often, more than one contact to the same client was required, in order to access if the product (bank term deposit) would be ('yes') or not ('no') subscribed.

The data is publicly available: Bank Marketing. Dua, D. and Graff, C. (2019). UCI Machine Learning Repository. Irvine, CA: University of California, School of Information and Computer Science.

The model repository is publicly available at: bank_marketing repository.

The Model Card also available at: model folder in bank_marketing repository.

The classification goal is to predict if the client will subscribe (yes/no) a term deposit (variable y). You can access the documentation and test the API from the link: Documentation and API.
